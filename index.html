О компании

Инженерно-технологическая компания, специализирующаяся на разработке и внедрении отечественных решений для управления производственными процессами. Тренажёрный комплекс

Астра СИМ 2.0

Астра СИМ 2.0 — тренажёрный комплекс, позволяющий проводить обучение и отработку практических навыков оперативного персонала, оценку его профессионального уровня.

ТК реализует следующие бизнес-функции:

  • Моделирование физико-химических процессов
  • Моделирование алгоритмов систем управления
  • Работа в автоматическом и ручном режимах
  • Проверка теоретических знаний персонала
  • Формирование отчётных документов

Функциональные характеристики программного обеспечения

Программное обеспечение «Астра СИМ 2.0» предназначено для обучения операторов АСУ ТП (децентрализованной системы управления) промышленных объектов.

ПО «Астра СИМ 2.0» решает следующие задачи:

  • Освоение технологии производства, интерфейса оператора
  • Отработка действий в среде управления при пуске и останове установки
  • Управление энергоэффективностью операторов, его навыков и опыта работы
  • Объективная оценка уровня подготовки и компетенции операторов

Архитектура ПО «Астра СИМ 2.0» построена с использованием микросервисного подхода

Сервис «Моделирование» • Сервис «Сценарии» • Сервис «Управление сессией обучения» • Сервис «Интеграция с системой управления»

Автоматизация КИП

Астра КИП

Астра КИП — предназначена для автоматизации повседневной деятельности эксплуатационного персонала, связанной с обеспечением надёжности, обслуживанием и ремонтом КИП.

ТК реализует следующие бизнес-функции:

  • Ведение базы данных технологического оборудования и средств измерений
  • Автоматическое формирование графиков ППР, технического обслуживания
  • Планирование сроков проведения МО, контроля, испытаний
  • Учёт актуализации данных о датчиках и регуляторах

Функциональные характеристики программного обеспечения

Программное обеспечение «Астра КИП» предназначено для автоматизации деятельности служб КИП промышленных объектов.

ПО «Астра КИП» решает следующие задачи:

  • Централизованное управление базой данных оборудования, его состояния и месте установки
  • Контроль состояния устройств и выполнения диагностики на лету
  • Проведение удалённой настройки приборов
  • Сбор детальных сводных отчётов об общей совокупной готовности КИП

Архитектура ПО «Астра КИП» построена с использованием микросервисного подхода

Сервис «База данных оборудования» • Сервис «Планирование настройки и диагностики» • Сервис «Документация»
